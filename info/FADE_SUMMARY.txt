## БЫСТРЫЙ ОТВЕТ

**Q**: Почему fade in/out увеличивают h1 на ~^2?

**A**: Это не bug, это математическое следствие FFT convolution с finite signals. 
- Fade создает discontinuities между сигналом и zero padding
- Эти разрывы интерпретируются как импульсы при deconvolution
- Два разрыва (начало и конец) взаимодействуют → артефакты энергии

**Решение**: Не применять fade к fundamental sweep, применить только к harmonics.

## КЛЮЧЕВЫЕ ВЫВОДЫ

### 1. Что происходит?
- ✓ Fade создает discontinuities на границах сигнала
- ✓ FFT convolution предполагает периодические граничные условия
- ✓ Discontinuities → импульсы при deconvolution
- ✓ Два разрыва (начало + конец) взаимодействуют

### 2. Почему энергия увеличивается?
- ✓ Свертка двух разрывов → квадратичный эффект
- ✓ Фазовая и envelope компенсация снижает фактор с 4x до ~1.16x
- ✓ Результат: peak увеличение на ~16%

### 3. Почему это проблема?
- ✓ Искусственное увеличение h1 → неправильное измерение THD
- ✓ Edge artifacts → порчит спектр в начале и конце
- ✓ Это систематическая ошибка, не случайный шум

### 4. Почему решение работает?
- ✓ Убираем fade из signal (inverse filter) → нет discontinuity
- ✓ Harmonics все еще имеют fade → спектральная гладкость OK
- ✓ Деконволюция становится чистой → чистый результат

### 5. Это bug метода?
- ✗ **НЕТ** — это математическое следствие FFT convolution
- ✓ Правильное понимание лежит в основе РЕШЕНИЯ
- ✓ Используя fade правильно → получаем правильный результат

## РЕКОМЕНДАЦИЯ

### ✓ ПРИМЕНИТЬ ВАРИАНТ 1

**Код изменения** (в farina2.py или gui.py):

```python
# ДО (проблемное):
signal = np.sin(2 * np.pi * f_start * K * (np.exp(t / K) - 1))
signal[:fade_length] *= np.linspace(0, 1, fade_length)      # УДАЛИТЬ ЭТУ СТРОКУ
signal[-fade_length:] *= np.linspace(1, 0, fade_length)     # УДАЛИТЬ ЭТУ СТРОКУ

# ПОСЛЕ (исправленное):
signal = np.sin(2 * np.pi * f_start * K * (np.exp(t / K) - 1))
# NO FADE on fundamental!
```

**Benefit**:
- ✓ Peak в h1 уменьшается на ~16%
- ✓ Edge artifacts уменьшаются на ~85%
- ✓ THD measurement становится точнее
- ✓ Спектральная гладкость не меняется (fade еще на harmonics)
