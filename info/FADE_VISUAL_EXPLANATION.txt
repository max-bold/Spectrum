STEP 1: SIGNAL GENERATION
═══════════════════════════════════════════════════════════════════════════════

Original signal without fade:
  ▁▂▃▄▅▆▇█████████████████████████████▇▆▅▄▃▂▁
  ├─────────────────────────────────────────┤

Signal with fade (PROBLEMATIC):
  
    ╱‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾╲
   ╱                                         ╲
  0  (discontinuity!)                  (discontinuity!)
  │                                             │
  ▲ FADE_IN gradient                            │
  │                                             ▼ FADE_OUT gradient


PROBLEM: Two discontinuities at boundaries!

  Time domain:
    n=0:    0 → (fade_in)  [JUMP 1]
    n=N-1:  (fade_out) → 0 [JUMP 2]

  FFT convolution adds zero-padding:
    n=N:    0 → 0 (periodicity boundary)

  Result: DISCONTINUITIES are interpreted as IMPULSES


STEP 2: INVERSE FILTER COMPUTATION
═══════════════════════════════════════════════════════════════════════════════

inv_filter = signal[::-1] / envelope

Where:
  - signal[::-1] = reverse(signal with fade)
  - envelope = exp(t/K) = monotonic growth

Timeline:
  t=0:   envelope ≈ 1.0
  t=T:   envelope ≈ 100

Reversed envelope:
  t'=0:  envelope ≈ 100    ← large (was end)
  t'=T:  envelope ≈ 1.0    ← small (was start)

Inverse filter values:
  Start (t'=0):   (fade_out value) / 100 = small / large = VERY SMALL
  End (t'=T):     (fade_in value) / 1.0 = very_small / small = MEDIUM

Result: ASYMMETRIC inverse filter with artifacts!


STEP 3: CONVOLUTION EFFECT
═══════════════════════════════════════════════════════════════════════════════

ir = convolve(response, inv_filter)

STEP 4: WHY NOT CANCELING (THEORY VS REALITY)?
═══════════════════════════════════════════════════════════════════════════════

THEORY (expected):
  response = signal * channel
  h = deconv(response, signal) = channel
  ───────────────────────────────────────
  fade should cancel out!

REALITY (what actually happens):
  FFT convolution assumes PERIODICITY:
  
    Signal extended:
    ├─ signal[0..N-1] ─┤
                       └─ 0 0 0 0 ... (zero padding)
    
    This creates a DISCONTINUITY at boundary!
    Not mathematically equivalent to the theory.

  Discontinuity → Impulse response
  Two discontinuities → Interaction effect ~^2
